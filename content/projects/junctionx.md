Winning team at [JunctionX Málaga 2023](https://www.pta.es/noticias/malaga-techpark-acoge-el-junctionx-malaga-el-hackathon-tech-mas-grande-de-europa/)!

![Snapguard](/images/snapguard_banner.png)

We Developed a steganography-based fingerprinting tool to help schools track the misuse of sensitive photos. Each image embedded invisible metadata: school, timestamp, permissions, and a unique parent ID, letting us identify the exact source if a photo was shared against policy.

You can check the source code [here](https://github.com/Matesanz/junction-move38).
