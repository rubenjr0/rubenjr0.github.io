Encryption and signing of text and files using state-of-the-art algorithms. Everything runs 100% in the browser via WebAssembly, or in your terminal with the native Rust implementation.

You can [use Veil here](https://veil.rubenjr0.eu), or check the source code for the underlying library [here](https://codeberg.org/VeilE2EE/veil).
